# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in DermaAssistAI, please report it
**privately** — do not open a public issue.

- Use GitHub's [private vulnerability reporting](https://github.com/srgnunlu/dermaai/security/advisories/new), or
- Contact the maintainer at [@srgnunlu](https://github.com/srgnunlu).

Please include steps to reproduce and the affected version or commit. Reports will
be acknowledged as soon as possible, with updates as a fix is prepared.

## Scope & data safety

DermaAssistAI is research/educational software and is **not** a medical device.
Never include real patient data or personal health information (PHI) in reports,
test cases, or screenshots — use anonymized or synthetic data only.

## Supported versions

This is an early-stage project under active development; security fixes target the
latest `main`.
