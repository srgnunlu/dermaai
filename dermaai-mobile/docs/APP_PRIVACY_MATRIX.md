# App Store Privacy Matrix

Corio Scan does not use user images or health-related data for model training or product AI improvement.

| Data type | Collected | Linked to user | Tracking | Purpose |
| --- | --- | --- | --- | --- |
| Email address | Yes | Yes | No | Account authentication and support |
| Name | Optional | Yes | No | Account profile |
| User ID | Yes | Yes | No | Authentication, subscription, and app functionality |
| Skin-area photos | Yes | Yes | No | User-requested preliminary assessment and history |
| Health/context answers | Optional | Yes | No | User-requested preliminary assessment |
| Purchase history/subscription state | Yes | Yes | No | In-app purchase entitlement |
| Push token | Optional | Yes | No | App notifications |
| Diagnostics/log data | Limited | May be linked | No | Security, reliability, and support |

Service providers/processors may include OpenAI, Google Gemini, hosting/database providers, Cloudinary or local hosting, RevenueCat/Apple payments, and Expo push services. Confirm the active production provider list before submission.

Retention follows the active privacy policy and user settings. Account deletion removes account-owned database records, push tokens, tracking data, and supported uploaded files. Apple subscriptions must be cancelled separately by the user.
